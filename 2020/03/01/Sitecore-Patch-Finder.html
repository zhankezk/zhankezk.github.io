Sitecore Patch Finder

1 March, 2020
I built this little website a long time ago to search through the Sitecore Support Github (https://github.com/SitecoreSupport) repos to find bugs, patches and hotfixes. Only shared it with a few people but recently saw another similar website was released for precisely the same purpose, but mine is much better.

So I did some polishing and released mine on http://sitecorepatchfinder.com/.

It runs daily and converts all the patches in Sitecore Support repos into Azure Cognitive Search, and the website use search on top of the indexes.

Still polishing here and there but I guess use if you need it. 

And always, contact Sitecore Support before you apply any patches!
