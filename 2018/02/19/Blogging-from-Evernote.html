Blogging from Evernote

19 February, 2018
This is the first post that’s pulled from Evernote. Since I have started to maximise the use of Evernote, keeping notes of everything I do, so setup to pull content from Evernote directly to generate the website.

The current engine stay the same and can still work independently, which is a static website generator that auto commit to GitHub pages. The new component pulls content from Evernote and transform them into post files and replace all existing posts. 

Not perfect yet since it does a full sync every time, will setup web hooks when I am free so it’s smarter.

Most of the time, I am just too lazy to format and upload images again ;-)
